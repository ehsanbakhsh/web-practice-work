String

  1. Javascript strings are used for storing and manipulating text.
  2. String is zero or more chracters.
    var a = "Hello world";
  3. String is zero or more chracters written inside quotes.
    var name1 = "Ehsan";
    var name2 = 'Bakhsh';
  4. String has built-in length property which return length of chracter in string.
    var a = "Hello world";
    alert(a.length);
