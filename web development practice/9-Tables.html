Tables

Here we can discuss about tables.
First we add a tag of (table).In the opening and closing tag of table,we can two more tags of (thead) that means table header,and (tbody) that means table body
IN the tag of (thead) we can use a tag of (tr) that means tables row,and in the tag of (tr) we can use another tag of (th) that means here we can write any text that we can use as heading in table.
In the tag of (tbody) here we can aslo use the tag of (tr),and in this tag we can use a tag of(td), and here we can write our information about heading.
