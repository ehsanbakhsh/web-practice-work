Arrays:

  1. Javascript values are used to store multiple values in a single variable.
  2. An array is used to store a collection of data.
  3. It is an ordered colllection which store elements in sequence.
  4. We can use array to store list of something like this:
    1. Students
    2. Cars
    3. Food items
  5. If you want to store temperature of last 7 days in a variable, then you have to create 7 variables like that;
    • var mondayTemperature = 35
    • var tuesdayTemperature = 37
    • var wednesdayTemperature = 33
    • var thursdayTemperature = 27
    • var fridayTemperature = 25
    • var saturdayTemperature = 40
    • var sundayTemperature = 35
  6. Now what if you want to store temperature of morning and evening for 7 days. That will be 14 variables.
  7. How about temperature for a year morning and evening that 730
  8. It will be very difficult to manage and assign names to these variables.
  9. If you want to find out all days temperature was above 30 then it will be quite difficult.
  10. If you want to sort temperature in ascending or descending order that will not be possible with seperate multiple variables.
  11. With arrays you can create single variables and hold all teemperatur in it.
  12. With array you will be able to find and sort temperature easily.e.g:
    • var temperature = [34, 15 , 27, 65, 34, 28, 20]

Creating an array

  1. Creating an array using array literal.
    • var food = ["Burger" , "Pizza", "Sandwich", "Snacks"];
  2. Creating arrays using new kewyword.
    • var foods = new Array("Burger" , "Pizza", "Sandwich", "Snacks");
  3. Both are same, First one is more recomended way to create array.
  4. Array ccan be created for all datatypes or you can mix them in a single array.e.g:
    • var arr1 = ["Hello", "World", "Bye"];
    • var arr2 = [29, 38, 16, 22];
    • var arr3 = [true, false, true, false, false];
    • var arr4 = [23.2, 45.8, 98.12];
    • var arr5 = [{name : "John"}, {name : "Jason"}];

Accessing Array Elements:

  1. You access an array element by reffering to the index number.
  2. To access elements you provided number in square brackets.
    • var foods = ["Pizza", "Burger", "Snacks"]
      foods[0]; //Pizza
      foods[1]; // burger
      foods[2]; // Snacks
  3. Store results in variable or show output directly:
    var foods = ["Pizza", "Burger", "Snacks"]
    var a = foods[0]; // Pizza
    var b = foods[1]; // Burger
    var c = foods[2]; // Snacks
  4. Range of array index is from to 0 Array's length -1.
  5. First element is on 0 index.
  6. Second element on index 1.
  7. Third on index 2
  8. Last element will be on arrays length -1.e.g;
    1. Array has 5 elements then last element is on 4th index.
    2. Array has 8 elements then last element is on 7th index.

Accessing full array:

  1. The full array can be accessed by reffering to the array name.
    • var foods = ["Pizza", "Burger" , "Snacks"];
      console.log(foods); // Pizza, Burger, Snacks

Accessing index that does not exist.

  1. If you create array with elements 3 and try to access 4th element, it will return undefined
  2. There will be no-error in accessing index that doees not exist.
    • var foods = ["Pizza", "Burger", "Snacks"];
      console.log(foods[2]); // Snacks
      console.log(foods[3]); // undefined
      console.log(foods[8]); // undefined
