Splice Function.

  1. To add elements in array we have used:
    1. Push function-----Add element in last.
    2. Un-shift function-----Add element in start.
    3. Index-----Add element in last or replace existing element.
  2. If we want to add element in middle of array on any index mother then first/last we can use splice function.
  3. Splice function can add one or more element on particuler index in array.
  4. Splice function can replace oe or more element on particuler index in aray.
  5. Splice function returns element which removed from array, if no element removed then eturns empty array.
