Assignment Operaters

  1. Assignment operater assign value to variabls.
  2. When you need to apply arithmatic operation and assign value to same variable then you can also use them.
    • var a = 5;//equals = is assignment operater.
    • a = a + 2;//assign7 in variable a.
    • OR
    • var a = 5;
    • a+ = 2;//assign 7 in variable a.

a / = 5; a = a +/ 5;
