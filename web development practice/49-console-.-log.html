Console.log

  1. console.log is a function that write messages on console/terminal.
  2. Objective of console.log is to create logs for debugging.
  3. Instead of displaying text to user it shows output in browser's developer tool.
  4. Also with working with server side javascript, we can use console.log for loging and output will be in terminal.
console.log("Hello world");
console.log(2+3);

In our console window a word Hello world and 50is shown.


Press f12 for move to console window or right click on page and next press inspect and next open a window on the tab and next click on console option.
