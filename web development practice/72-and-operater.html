&& Logical operater

  1. This logical operater is used with two or more values (operands), and only evaluates to true if all the operands are true. It will return false if at-least one value is false.
    • alert(true && true) // Return true
    • alert(true && false) // Return false
    • alert(false && true) // Return false
    • alert(false && false) // Return false
  2. Examples:
    • var a = 70;
    • var b = a > 50 && a < 80
    • alert(b);// return true

    • var c = 90;
    • var d = a > 50 && a < 80
    • alert(b);// return false
