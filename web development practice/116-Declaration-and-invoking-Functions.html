Functions Decleration:

  1. A javascript is deined with the function keyword, Followed by a name,Followed by parenthesis().
  2. Function names can contain letters, digits, underscores, and dollar signs(Same rules as variables.)
  3. The parantheses may include parameter names seperated by commas:(parameter1,parameter2,...).
  4. The code to be executed, by the function, is placed inside curly brackets:{}.
    • function name (parameter1 , parameter2){
      //Code to be executed
      }
    • function sum(a,b){
      return a + b;
      }
  5. Declared functions are not executed immediatly.They are "Saved for later use", and will be executed later,when they are invoked(Called upoon).

Invoking a Function:

  1. Functions execute when the function is called.
  2. This process is called invocation.
  3. You can invoke a function by referencing the function name,Followed b an open and closed parenthesis:().

Decleration Example:

  • function showMessage(Message){
    console.log(message);
    }

Invoking Example:

  • showMessage("Hello World");

Parameters vs Arguments:

  1. Parameters:
    1. Function parameters are listed inside the parenthesis () in the function defination
  2. Arguments:
    1. Function arguments are the values recieved by the function when it is invoked.
