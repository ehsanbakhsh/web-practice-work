Increment and Decrement Operaters.

  1. WHile working on application you will frequently required to increase variable by 1 or decrease vaiable by 1.
  2. For this situation you can use increment and decremen operaters.
  3. ++ incement operater
  4. -- decrement operater
  5. These operaters are used as prefix and postfix.

Diference between increment nad decrement operater.

  1. You wil not find anydifference if ou will not assign result of prefix and postfix to any other variable.
  2. Prefix operaterfirst increase/Decrease the value in variable and then assign result to other variable.
  3. Postfix operater first assign the value in other value then increase/decrease value in actual variable.
