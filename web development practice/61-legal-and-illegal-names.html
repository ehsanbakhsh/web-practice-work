Legal and Illegal variable names

  1. A variable name can't contain any space
  2. A variable name can contain only letters, numbers, dollar sign and underscoes.
  3. The first chracter must be a letter. or an underscore(_) or a dollar sign($).
  4. Subsequent chracters may be letters, digits, underscores or doller signs.
  5. Numbers are not allowed nas the first chracter of variable.

Legal Names;

  • var Hello = 56;
  • var _xyz = 44;
  • var $work = 90;
  • var user2 = 65;
  • var i_info = 99;
  • var my$work = 77;

Illegal names;

  • var 2user = 12
  • var my user = 23
  • var hello#world = 98
  • var my-info = 77
  • var my?info = 55
  • var my*info = 45
