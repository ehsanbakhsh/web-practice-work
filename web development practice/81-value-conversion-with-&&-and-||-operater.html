Value conversion with && operater.

  1. var a1 ='cat' && 'dog';// t && t returns dog
  2. var a2 =false && 'cat';// f && t returns false
  3. var a3 ='cat' && false;// t && f returns false
  4. var a4 = '' && false;// f && f returns ''
  5. var a5 =false && '';// f && f returns false
  6. var a6 =0 && 1;// f && t returns 0

Value conversion with || operater.

  1. var a1 ='cat' || 'dog';// t && t returns cat
  2. var a2 =false || 'cat';// f && t returns cat
  3. var a3 ='cat' || false;// t && f returns cat
  4. var a4 = '' || false;// f && f returns false
  5. var a5 =false || '';// f && f returns ""
  6. var a6 =0 || 1;// f && t returns 1
