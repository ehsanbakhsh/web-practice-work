Recursive function:

  1. A recursive function is a function that calls itself.
  2. Recursion is a technique for iterating over an operation by having a function call itself repeatedley until it arrives at a result.
  3. In some ways, recursion is analogous to a loop. Both execute the same code multiple times, and both requires a condition(to avoid an infinite loop, or rather, infinite recursion in this case).
  4. The classic example of a function whee recursion can be applied is facorial.
  5. Factorial of a number n can be defined as product of all positive numbers less then or equal to n.
  6. It is the multiplying sequence of a number in descending order till 1. Ot is defined by symbol of exclaimation(!).
  7. E.g : Factorial of 6 is:
    6 x 5 x 4 x 3 x 2 x 1 = 720
