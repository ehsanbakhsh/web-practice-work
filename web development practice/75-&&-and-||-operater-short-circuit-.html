Why && and || operater are called short circuit?

  1. && and || operater stops evaluation of epression once they find desired value.
  2. && stops evaluaion as soon as it finds false and returns false.
  3. && returns true if all values are true.
  4. || stopss evaluation as soon as it finds true and returns true.
  5. || returns false if all values are false.
  6. Examples of && operater.
    • //Returns false, evaluation stops at first value.
    • var a = false && true && false;
    • //Returns false, evaluation stps at last value.
    • var b = true && true && false;
    • //Returns false, evauation stops at second value.
    • var c = true && false && true;
  7. Examples of || operater.
    • //Returns false, evaluation stops at second value.
    • var d = false && true && false;
    • //Returns false, evaluation stps at first value.
    • var e = true && true && false;
    • //Returns false, evauation stops at last value.
    • var f = true && false && true;
