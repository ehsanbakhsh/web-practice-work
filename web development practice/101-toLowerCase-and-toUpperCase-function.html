String Functions:

During development you will often require to manipulate string and for that string has many functions to do the job.

  1. toLowerCase()
  2. toUpperCase()
  3. slice()
  4. IndexOf()
  5. lastIndexOf()
  6. charAt()
  7. replace()
  8. split()
