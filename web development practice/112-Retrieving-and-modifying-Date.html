Retrieving and Modifying date:

  1. Once we have a date, we can access all the components of the date with various built0in methods.
  2. The method will return ach part of the date relative to the local timezone.
  3. Each of these methods starts with get, and will return the relative number.

Date/Time Method Range Example
Year getFullYear() YYYY 1970
Month getMonth() 0 - 11 0 = Jnauary
Day(of the month) getDate() 1-31 1 = 1st of the month
Day (of the week) getDay() 0 - 6 0 = Sunday
Hour getHours() 0 - 23 0 = Midnight
Minute getMinutes() 0 - 59 ........................
Second getSeconds() 0 - 59 ........................
Milliseconds getMilliseconds() 0 - 999 ........................
Timestamp getTime() Milliseconds since epoch time ........................

Retrieving the Date component:

var date = newDate("June 14 2019 10:45:25");

Method Returns
date.getFullYear(); // 2021
date.getMonts(); // 5
date.getDate(); // 14
date.getDay(); // 5
date.getHours(); // 10
date.getMinutes(); // 45
date.getSeconds(); // 25
date.getMilliseconds(); // 1560491125000

Modifying the Date

  1. For all the get methods That we learned,there is a corresponding set method.
  2. Where get is used to retrieve a specific component from a date, set is used to modify component of a date.
