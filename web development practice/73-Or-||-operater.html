OR Operater

  1. The logical operater || (OR) is used with two or more values, but it evaluates to true if any of the operand are true, so only evaluates to false if both operands are false.
  2. Examples:
    • alert(true || true); // Return true
    • alert(true || false); // Return true
    • alert(false ||true); // Return true
    • alert(false || false); // Return false

  3. || Logical operater

    • var a = 60;
    • var b = a < 50 || a > 70;
    • alert(b);// return false

    • var c = 80;
    • var d = c < 50 || c > 70;
    • alert(d);// Return true
