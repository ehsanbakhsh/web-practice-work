Logical Operaters

  1. Logical operaters are used to determine logics between variables and values.
  2. Logical opeaters required boolean operands on both sides of operater.
  3. However, the && and || operaters actually return the value of one of the specified operands, so if these operaters are used with non-boolean values, they will return a non-boolean value.
  4. There are three logical peraters in javascript:
    • || Or operater
    • && And operater
    • ! (NOT)
