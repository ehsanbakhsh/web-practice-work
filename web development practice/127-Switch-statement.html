Switch Statement:

  1. The switch statement executes a block of code depending on different casses.
  2. The switch statement is a part of javasacript's "conditional" statements which are used to perform different actions based on different conditions.
  3. Switch statement works for equality checks only, you can not apply range greater then or less then checks.
  4. Tis is how it works:
    1. The switch ststement evaluates an expression.
    2. The value of expression is then compared with the values of each case in the structure.
    3. If there is a match, the associated cpde of block is executed.
  5. The switch has one or more case blocks and an optional default.

Switch - Grouping of case:

  1. Sometimes you will want differrent casses to use the same code, or fall through to a common default.
  2. If you skip the break and expression match to the case where there is no break thebn it will also fall-through the next case.

Switch-Strict comparison:

  1. Switch casses uses strict comparison(===).
  2. The values must be of the same type to match.
  3. A strict comparison can onlu be true if the operands are of the same type.
