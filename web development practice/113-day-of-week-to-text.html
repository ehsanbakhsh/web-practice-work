Day of week to text:

  1. getDay() function returns day of week, but it will give numbers representing aday from 0 to 6.
  2. 0 represents sunday. 1 represents monday and soo on.
  3. If you want to convert this value into text representation then you have to do it yourself.
  4. You create array represnts day of week in text format and then map value from getDay() function to array.
