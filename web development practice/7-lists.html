There are two types of lists that we discuss here:

  1. Ordered List
  2. Un-ordered List

To create ordered lists we can use a tag of (ol) that is known as Ordered list.
in every list we can use a tag of (li) that is known as List Items.

Ordered list can provide numbers or integers with all list items.Here is an example of Ordered list

ODERED List

  1. This is the first list item.
  2. This is the second list item.
  3. This is the third list item.
  4. This is the fourth list item.
  5. This is the fifth list item.

To create un-ordered list we can use the tag of (ul) that meansUN-ORDERED LIST
In the un-ordered list we can also use the he tag of (li) that means LIST ITEMS

Un-ordered lists can't provide numbers.unordered list can show bullets.Here is an example of un-ordered list

UN-ORDERED List

  • This is the first list item.
  • This is the second list item.
  • This is the third list item.
  • This is the fourth list item.
  • This is the fifth list item.
