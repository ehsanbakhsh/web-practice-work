Value conversion to boolean part 1.

  1. I javascript value or expression can be converted to boolean.
  2. If you apply boolean comparison on values then javascript convert the value into boolean and make comparison.
  3. You can use the values in if/else condition and they will result in true or false according to values.
  4. Following values will be converted to false:
    1. null
    2. NaN
    3. 0
    4. "" or ''
    5. undefined
  5. All else will be conveted to true.
  6. Following values wil be converted to true.
    1. {} // object
    2. "Any text" // String with any text.
    3. 1 // Any number other then 0
