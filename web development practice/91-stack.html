Stack

  1. Stack is a linear data structure represented by a real physical stack or pile where insertion and deletion of items takes place at one end called top of the stack.
  2. The basic concept can be illustrated by thinking of your data set as a stack of plates or books where you can only take the top item of the stack in order to remove thingd from it.
  3. The basic implementation of stack is also called LIFO (Last in First out).
  4. To behave as a stack you have to use push and pop funstions on array.
  5. Push function will be add elements at the end in array.
  6. Pop function will remove and return last element from array.
