For loop.

  1. Loop in programming languauges is a feature which facilitates the execution of a set of instructions/functions repeatedley while some conditions evaluates to true.
  2. The far statement creates a loop that is executed as long as a condition is true.
  3. It will only stop when the condition becomes false.
    • for (initiallization; condition; expression){
      //code to be executed
      }
  4. Initialization is done by (one time) before the execution of block code.
  5. Condition for executing the block code and exit loop
  6. Expression is executed (every time) after the code block is executed.

Examples:

  1. for (var i = 0; i < 3; i ++){
    console.log(i);
    }
  2. Output:
    • 0
    • 1
    • 2
