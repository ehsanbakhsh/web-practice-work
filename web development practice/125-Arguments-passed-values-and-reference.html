Argument passed value and reference:

  1. Primirive data is passed by value:The functon only gets to know the values, not the argument's location.
  2. If a function changes an argument's value,it does not change the parameter of original value.
  3. Changes to arguments are not visible(Reflected) outside the function.
  4. In javascript, object references are values.
  5. Non-primitive value such as Array or a user defined object are passedby reference.
  6. If function changes the objects properties, thar change is visible outside the function.
