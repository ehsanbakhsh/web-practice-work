Not Operater

  1. Using the ! operater in front of a boolean will convert it to opposite value. It means that a true value will return false, And a false will return true. This method is known as negation.
  2. Examples:
    • alert(!true) // false
    • alert(!false) // true

    Logical NOT

    • var a = 60;
    • var b = !(a < 50)
    • alert(b);// return true

    • var c = 80;
    • var d = !(c > 50);
    • alert(d); // return false
