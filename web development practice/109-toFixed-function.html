toFixed() Function (Controlling the length of decimal):

  1. In arithmatic operation you may face numbers with many decimal places.
    var average = (15 + 23 + 39) / 3; // 25.666666666
  2. To limit decimal places to specified number you can call toFixed() function on number and round last digit.
    var avg = average.toFixed(3); // returns 25.667.
