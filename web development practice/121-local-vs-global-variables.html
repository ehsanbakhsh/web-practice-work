Local vs Global variables:

  1. Variables can have local and global scope.
  2. A global variable is one that's declared in the main body of your code, not inside a function.
  3. A local variable is one that's declared inside a function.
  4. A local variable can be either a parameter of the function, which is declared implicityby being named as a parameter, or a variable declared explicity in the function with the var keyword.
  5. Global variable is meaningful in every secton of your code, weather that code is in the main body or in any of the functions.
  6. Local variable is the one thats meaningful only within the function that declares it.
  7. There are two differences between global and local variables where they're declared, and where they're known and can be used.
    8. Global Variables Local variables
    0 Declared in the main code. Declared in a function.
    02 Known everywhere, Useable everywhere. Known only inside the function, useable only inside the function.
