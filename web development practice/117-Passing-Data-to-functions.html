Passing data to functions:

  1. In order for a function to become a programmable robot rather than a one-job robot, you have to set it upto recieve the data you are passing.
  2. You can pass any type of data to function depending on requirment.
  3. Example:
    1. function multiply(num1 , num2){
      var num3 = num1 * num2;
      console.log("Num3 " num3);
      }
      multiply("3,6");
      multiply("4,2");
    2. function showMessage(name){
      console.log("Hello " , name);
      }
      showMessage("Mike");
      multiply("John");
