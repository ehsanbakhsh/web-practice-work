replace() Function

  1. The replace() function replaces a specified vslue with another value in a string
  2. The replace() function does not change the string it is called on. It returns a new string.
    • var a = "To be or not to be";
    • var b = a.replace("be" , "Hello") // return "To Hello or not to be"
  3. By default, the replace() function replaces only the first match
  4. To replaces all matches, use a regular expression with a /g flag (Global match) and without quotes.
    • var a = "To be or not to be";
    • var b = a.replace(/be/g , "Hello") // return "To Hello or not to Hello"
