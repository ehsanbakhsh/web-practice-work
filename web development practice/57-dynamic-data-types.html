Dynamis Data types

  1. Javascript has dynamic types. This means that are the same variable can be used to hold different datatypes.
  2. You can assign any type of valuein variable and anytime datatype the datatype of variable will be changed.
    • For example
    • var name = "Ehsan"//It's String.
    • name = 25;//It's Number.
    • name = true;//It's Boolean.
