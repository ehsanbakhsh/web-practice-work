While loop:

  1. In this example, The code in the loop will run, over and over again as log as the variable (i) is less then 10:
    • while (i < 10){
      console.log("I "+i);
      i++
      }
  2. You will use while loop when execution is dependent on user's input.

Do/while Loop:

  1. The do/while loop is a variant of thr while loop which executs the code block once, before checking if the condition is true, thhen it will repeat the loop as long as the condition is true.
  2. You will use do/while loop when execution is dependent on user input but it needs to run code block at least once.
    • Syntex:
      do{
      // Code block to be executed
      }
      while (condition);
  3. Example:
    do{
    console.log("I "+i);
    i++
    }
    while (i < 10);
