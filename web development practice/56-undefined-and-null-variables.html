Differance between null and un-defined.

  1. Un-defined means a Variable ha been declared but has not yet been assigned a value.
  2. null is an assignment value. It can be asigned to a variable as a representation of no-value.
  3. Un-defined and null are two disinct types: Un-defined is a type itself (undefined) while null is an object.
  4. Unassigned variables are initialized by javascript with a default value of undefined. Javascript never sets a value of un-defined. Javascript never sets the value of null. That must be done programmatically.
