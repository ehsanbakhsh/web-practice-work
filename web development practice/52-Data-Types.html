Data Types

  1. Variables contains data
  2. Each data has type that's why we call it data type.
  3. Variables can hold numbers like 100 and text values like Ehsan.
  4. In programming language, text values are called stings.

Names of data types.

There are 7 types of data types.

  1. Six types are considered to be primitives.
    1. Number -- integers, floats etc
    2. String -- An array of chracters.
    3. Boolean -- True or False
    4. null -- No Value
    5. Un-Defined -- A declared value but hasn't been given a value.
    6. Symbol -- A unique value that's not equal to any other value.
  2. Comples Types
    1. Objects
    2. Functions

      3. Functions are callable object that executes a block of code, it lies under the object type.
