Conditons if-else

  1. Up-until, all the code in our programms has been executed chronologically.
  2. Very often when you write code, you want to perform different actions for different decisions.
  3. You ca use conditional statements in your code to do this.
  4. In javascript we have following conditional statements:
    1. Use if to specify a block of code to be executed, if a specified condition is true.
    2. use else to specify a block of code to be executed, if the same condition is false.
    3. use else if to specify a new condition to test, if the first condition is false.
    4. use switch to specify many alternative blocks of code to be executed.(DISCUSSED LATER).

Condition : if

  1. The if statement is the fundamental control statement that allows javascript to make decisions and execute statement conditionslly.
    • if (condition) {
      // Block of code to be executed if the condition is true}
  2. Example:
    • var age = 12;
    • if (age > 9) {
      console.log("Age = "+age);
      }

Condition : else

  1. Use the else statement to specify a block of code to be executed if the condition is false.
    • if (condition) {
      //block of code to be executed if condition is true.
      }
    • else {
      //block of code to be ececuted if condition is false
      }
  2. Example:
    • var age = 15;
    • if (age > 18) {
      console.log("Qualifies for driving.");
      }
      else {
      console.log("Does not qualifies for driving.");
      }
