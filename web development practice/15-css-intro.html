What is css?

Cascading style sheet which is known as css is a style sheet languagethat used to handle the presentation of the web page containing Html.It makes our websites beautiful and modern looking.

  • Css stands for cascadin styles sheet.
  • Css gives styles to raw Html.
  • Css is used to gives styles to our web pages.
  • Css is used to make websites responsive.
  • Html is used tos tructure a website.
  • Css takes responsibility of design.

CSS SYNTEX......

SELECTOR

First of all we can write selecter. Selecter shows that where you can apply styles in html page.For Example:If i can apply styles on the paragraph or heading 1 , i can writ its tags before the curly brackets and apply css in curly brackets.e.g
p, h1, h2, h3{ }

  • Css selecters are used to be find the element whose property will bw set.
  • Selecters are used to target the html elements.
  • Selectors makes it easy for us to easily target single/multiple Html elements in the markup.

Types of basic css selecters:

  1. Css element selecter
  2. Css id selecter
  3. Css class selecter
  4. Css grouping selecter

PROPERTY

Property is that work we cxan do with the tag that is written before curly brackets. Fo Example: If i can chamge the color of all text present in the paragraph then i can apply css like this
e.g: p, h1, h2, h3 {color}
Here color is the property of p , h1 , h2 and h3.

VALUE

After the property we can set it's value For Example: If i can decide to change change the color of p , h1 , h2 , and h3 first i can write its property and add a colon : and the color name that we want such that blue and last we add semicolon ; .
It is writen like this:
p, h1, h2, h3 {color:blue;}

How many ways to add css in html?

There are three ways to add css in hypertext markup language (HTML)

  • Inline css
    • Css is added to the elements directly using style attributes.

  • Internal css
    • Css is kept inside the head tags in < style > tag

  • External css
    • Css is kept seperately inside a .css style sheet.
